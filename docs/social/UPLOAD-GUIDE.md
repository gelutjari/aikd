## 🔧 Manual Upload Steps

### Instagram Reels
1. Open Instagram app
2. Tap "+" → Reel
3. Upload `assets/demo-vertical.mp4`
4. Add caption from above
5. Add hashtags
6. Share

### TikTok
1. Open TikTok app
2. Tap "+" 
3. Upload `assets/demo-vertical.mp4`
4. Add caption from above
5. Choose trending sound
6. Post

### Threads
1. Open Threads app
2. Tap compose
3. Add video `assets/demo-square.mp4`
4. Add caption from above
5. Post

### Twitter/X
1. Open Twitter
2. Start thread with Tweet 1
3. Upload video/GIF
4. Continue thread
5. Tweet all
